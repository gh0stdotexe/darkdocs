> ## <span class=&quot;callout-icon&quot;>ðŸ“˜</span><p>Full Analytics</p>

<p>In order to see DNS queries in the Activity Log, you need to enable &quot;Full&quot; <Link href=&quot;/docs/analytics-levels&quot; target=&quot;&quot; title=&quot;&quot; class=&quot;doc-link&quot; data-sidebar=&quot;analytics-levels&quot;>Analytics log level</Link>.</p>

<p>Simply navigate to the Analytics -> <Link href=&quot;https://controld.com/dashboard/activity-log&quot; target=&quot;&quot; title=&quot;&quot;>Activity Log</Link> section, and choose a Device you wish to monitor. From here, you can filter the data in multiple ways:</p>

- **Date** - choose the date and time to view queries from the chosen time period 

- **Protocols** - show DNS queries that used specific DNS protocols (Legacy, DNS-over-HTTPS, etc)

- **Actions** - show DNS queries based on a performed action (block, bypass, redirect) 

- **Filters** - show DNS queries that were blocked by a specific Filter

- **Services** - show DNS queries that were acted upon by a specific Service

- **Location** - show DNS queries that were redirected to a specific location

## <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;real-time-logging&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-real-time-logging&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Real Time Logging</div><Link aria-label=&quot;Skip link to Real Time Logging&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#real-time-logging&quot;></Link>

<p>You can toggle the the switch in the top right corner to enable real time logging. This will show you DNS queries that are being made on a chosen device in real time.</p>
