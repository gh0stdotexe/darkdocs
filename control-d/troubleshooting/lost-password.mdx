## Lost Password Recovery

1. **Password Reset**
   - Visit the login page
   - Click "Forgot Password"
   - Enter your email address
   - Follow reset instructions

2. **Email Not Received**
   - Check spam folder
   - Verify email address
   - Add no-reply@controld.com to contacts
   - Contact support if needed

## Lost 2FA Access

### Lost Authentication Device
1. Use backup codes (if saved)
2. Contact support with account verification:
   - Account email
   - Organization name
   - Recent activity details
   - Payment information

### No Backup Codes
Contact support with:
- Government ID
- Account ownership proof
- Recent account activity
- Payment verification

## Prevention Tips

### Password Security
- Use a password manager
- Create strong passwords
- Enable 2FA
- Store backup codes safely

### 2FA Best Practices
- Save backup codes securely
- Use multiple 2FA methods
- Keep authenticator app backups
- Document recovery process

## Business Accounts

### Admin Recovery
1. Secondary admin can reset access
2. Use organization recovery process
3. Contact support with business verification

### Team Member Recovery
1. Admin can reset password
2. Admin can disable 2FA
3. User can follow standard recovery

## Security Recommendations

### After Recovery
1. Change password
2. Reset 2FA if compromised
3. Review account activity
4. Update security settings

### Future Prevention
1. Document recovery process
2. Store backup codes securely
3. Use multiple 2FA methods
4. Keep contact info updated

<Warning>
Never share your password or 2FA codes with anyone, including Control D support.
</Warning>

<Tip>
Store backup codes in a secure password manager or physical safe to prevent future lockouts.
