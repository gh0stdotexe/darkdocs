<header id="content-head"><div class="row clearfix"><div class="col-xs-9"># Legacy Resolver

<div class="excerpt"><div class="rm-Markdown markdown-body" data-testid="RDMD"><p>Issue IPv4 and IPv6 legacy DNS resolvers to use on older physical devices.</p></div></div></div><div class="col-xs-3">[<i aria-hidden="true" class="icon icon-edit"></i> Suggest Edits](/edit/legacy-resolver)</div></div></header><div class="grid-container-fluid" id="content-container"><section class="content-body grid-75"><div dehydrated="# <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;what-are-legacy-resolvers&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-what-are-legacy-resolvers&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>What are Legacy Resolvers</div><Link aria-label=&quot;Skip link to What are Legacy Resolvers&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#what-are-legacy-resolvers&quot;></Link>

<p>A Legacy Resolver is a UDP 53 based DNS resolver, and it&#x27;s just an IP pair. If you&#x27;re deploying Control D on a device that does not support Secure DNS protocols, you have to enable this feature. Once you do, you will be issued with 2 pairs of Legacy DNS resolvers: 2x IPv4 and 2x IPv6. </p>
## <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;example-devices&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-example-devices&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Example Devices</div><Link aria-label=&quot;Skip link to Example Devices&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#example-devices&quot;></Link>

<p>Here is a short list of operating systems and device types that require Legacy DNS resolvers</p>

- Windows 10

- Most routers

- Android TV boxes

- Smart TVs

## <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;how-to-use&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-how-to-use&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>How to Use</div><Link aria-label=&quot;Skip link to How to Use&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#how-to-use&quot;></Link>

<p>Simply edit an Endpoint and toggle the feature ON. Then view your resolver configuration to see your Legacy DNS IPs. </p>
