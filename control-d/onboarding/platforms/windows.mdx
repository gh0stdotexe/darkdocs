## Installation Methods

### GUI Setup Utility (Recommended)
1. Download the Control D Setup Utility from the dashboard
2. Run the installer as administrator
3. Follow the setup wizard
4. Select your preferred profile
5. Complete the installation

### Manual Configuration
1. Open Windows Settings
2. Navigate to Network & Internet > Network Adapter
3. Select your active connection
4. Click "Edit DNS server assignment"
5. Enter your Control D DNS resolver

## Advanced Configuration

### DNS-over-HTTPS
```powershell
# Enable DoH in Windows
Get-DnsClientDohServerAddress
Add-DnsClientDohServerAddress -ServerAddress "your-code.dns.controld.com" -AllowFallbackToUdp $True
```

### DNS-over-TLS
- Configure via PowerShell or Group Policy
- Requires Windows 11 or Server 2022
- Supports automatic fallback

### Multiple Profiles
- Create different network profiles
- Assign different Control D configurations
- Schedule profile switching

## Troubleshooting

### Common Issues
- DNS cache conflicts
- Network adapter settings
- Firewall blocking
- VPN interference

### Verification Steps
1. Check connection status
2. Verify DNS settings
3. Test resolution with `nslookup`
4. Check Windows DNS cache

## Enterprise Deployment

### Group Policy
- Configure via GPO
- Deploy settings domain-wide
- Manage multiple locations

### MDM Integration
- Microsoft Intune support
- Configuration profiles
- Remote management

<Tip>
Use `ipconfig /flushdns` to clear the DNS cache if you experience resolution issues after configuration changes.
