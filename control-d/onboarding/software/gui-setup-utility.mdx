<header id="content-head"><div class="row clearfix"><div class="col-xs-9"># GUI Setup Utility

<div class="excerpt"><div class="rm-Markdown markdown-body" data-testid="RDMD"><p>Optional setup utility that makes setup a breeze.</p></div></div></div><div class="col-xs-3">[<i aria-hidden="true" class="icon icon-edit"></i> Suggest Edits](/edit/gui-setup-utility)</div></div></header><div class="grid-container-fluid" id="content-container"><section class="content-body grid-75"><div dehydrated="<p>This graphical utility can be optionally used by end-users to configure a Control D <Link href=&quot;/docs/devices&quot; target=&quot;&quot; title=&quot;&quot; class=&quot;doc-link&quot; data-sidebar=&quot;devices&quot;>Device</Link> with 2 simple steps. Using this tool skips all the manual configuration steps and makes setup a breeze. </p>
# <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;download&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-download&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Download</div><Link aria-label=&quot;Skip link to Download&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#download&quot;></Link>

<p>It&#x27;s recommended you use the web tutorial part of the setup flow. It will prompt you to install the app when needed. </p>
# <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;create-a-device&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-create-a-device&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Create a Device</div><Link aria-label=&quot;Skip link to Create a Device&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#create-a-device&quot;></Link>

<p>Go into the Endpoints section, and <Link href=&quot;https://controld.com/control-panel/filters?deviceDialog=platform&amp;overlay=profile-management&quot; target=&quot;&quot; title=&quot;&quot;>Add an Endpoint</Link> for the appropriate platform. If you&#x27;re using Free DNS resolvers, click on &quot;Setup Guide&quot; button. </p>
# <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;get-configuration-code&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-get-configuration-code&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Get Configuration Code</div><Link aria-label=&quot;Skip link to Get Configuration Code&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#get-configuration-code&quot;></Link>

<p>The **Resolver ID** is the configuration code you&#x27;re looking for. </p>
<p><span aria-label=&quot;&quot; class=&quot;img lightbox closed&quot; role=&quot;button&quot; tabindex=&quot;0&quot;><span class=&quot;lightbox-inner&quot;><Image alt=&quot;&quot; loading=&quot;lazy&quot; src=&quot;https://files.readme.io/d33e076-image.png&quot; align=&quot;&quot; caption=&quot;&quot; height=&quot;auto&quot; title=&quot;&quot; width=&quot;auto&quot;/></span></span></p>
<p>If you don&#x27;t have the app installed yet, click **Automatic Setup** button.  You will be presented with this screen. </p>
<p><span aria-label=&quot;&quot; class=&quot;img lightbox closed&quot; role=&quot;button&quot; tabindex=&quot;0&quot;><span class=&quot;lightbox-inner&quot;><Image alt=&quot;&quot; loading=&quot;lazy&quot; src=&quot;https://files.readme.io/e890cd7-image.png&quot; align=&quot;&quot; caption=&quot;&quot; height=&quot;auto&quot; title=&quot;&quot; width=&quot;auto&quot;/></span></span></p>
<p>From here, you can download the utility app. </p>
# <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;paste-configuration-code&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-paste-configuration-code&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Paste Configuration Code</div><Link aria-label=&quot;Skip link to Paste Configuration Code&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#paste-configuration-code&quot;></Link>

<p>Paste the configuration code into the box of the app you just downloaded and launched. The interface looks similar on all supported platforms. </p>
<figure><span aria-label=&quot;1906&quot; class=&quot;img lightbox closed&quot; role=&quot;button&quot; tabindex=&quot;0&quot;><span class=&quot;lightbox-inner&quot;><Image alt=&quot;1906&quot; loading=&quot;lazy&quot; src=&quot;https://files.readme.io/ab4b42e-util.JPG&quot; title=&quot;util.JPG&quot; align=&quot;center&quot; class=&quot;&quot; caption=&quot;&quot; height=&quot;auto&quot; width=&quot;auto&quot;/></span></span><figcaption><p>This illustrates the Windows version, however MacOS, Android and iOS look virtually identical.</p></figcaption></figure>
<p>That&#x27;s it, once you see a success message you&#x27;re done here. The utility can be closed. Control D will run silently in the background (even after reboot), and use the DNS resolver associated with the device that was configured.</p>
<p>To verify that your system is configured correctly, you can go to the <Link href=&quot;https://controld.com/status&quot; target=&quot;&quot; title=&quot;&quot;>Status Page</Link>. </p>
## <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;modes&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-modes&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Modes</div><Link aria-label=&quot;Skip link to Modes&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#modes&quot;></Link>

<p>Depending on your OS, the Settings menu allows you to choose between up to 2 configuration modes. </p>

- **Managed ** (Default) - This mode uses <Link href=&quot;/docs/ctrld&quot; target=&quot;&quot; title=&quot;&quot; class=&quot;doc-link&quot; data-sidebar=&quot;ctrld&quot;>Command Line Daemon</Link> under the hood, to enable DNS-over-HTTPS using an installed system service. This will work in every operating system.

- **Native **- Configure DNS-over-HTTPS natively if your OS supports it (Windows 11, latest MacOS)

<p>We recommend you use the Managed mode, which is the default. </p>
