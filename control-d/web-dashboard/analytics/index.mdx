# <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;how-to-enable-analytics&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-how-to-enable-analytics&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>How to Enable Analytics</div><Link aria-label=&quot;Skip link to How to Enable Analytics&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#how-to-enable-analytics&quot;></Link>

<p>To enable Analytics, simply edit a Device and choose one of the <Link href=&quot;/docs/analytics-levels&quot; target=&quot;&quot; title=&quot;&quot;>Analytics Levels</Link>. Then head over to the Analytics Dashboard and view the stats.</p>
# <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;data-storage-rules&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-data-storage-rules&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Data Storage Rules</div><Link aria-label=&quot;Skip link to Data Storage Rules&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#data-storage-rules&quot;></Link>

## <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;storage-region&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-storage-region&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Storage Region</div><Link aria-label=&quot;Skip link to Storage Region&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#storage-region&quot;></Link>

<p>In Preferences -> Account, you can select a Storage Region. This is where your data will be physically stored should you choose to enable the Analytics feature.</p>
## <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;granular-data&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-granular-data&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Granular Data</div><Link aria-label=&quot;Skip link to Granular Data&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#granular-data&quot;></Link>

<p>The most granular data stored by Control D in any circumstance is kept for a maximum of three days. This is only the case when you have selected Full Analytics for the Device in question. In this case, Control D stores the following information about each of the DNS queries made against your Control D Deviceâ€™s resolvers:</p>

- Domain name

- Date and time of query

- Protocol used

- DNS return code

- Type of record

- Action taken (blocked, bypassed, or redirected)

- Reason for action taken (Filter, Service, Custom Rule, Default Rule)

- Context for reason (which specific Filter, Service, Custom Rule, and if redirected, which location or IP it was redirected via)

- Source IP address

- Destination IP address/es

<p>This data is what makes the Activity Log possible. We never store it unless you choose to select Full Analytics. It is impossible to see your browsing data in the Activity Log unless Full Analytics is selected for the Device being viewed for this exact reason: we respect you and your privacy.</p>
## <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;domain-data&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-domain-data&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Domain Data</div><Link aria-label=&quot;Skip link to Domain Data&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#domain-data&quot;></Link>

<p>The next category of data we can store with your explicit permission is the less granular data, kept for 1 month, which makes the Statistics: Full Analytics view possible. The granularity of this data is reduced to hourly, with no query timestamps. It comprises counts of domains resolved, actions taken, and associated metadata - like whether the action taken was due to a Custom Rule, Service, or Filter.</p>
## <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;counts&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-counts&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Counts</div><Link aria-label=&quot;Skip link to Counts&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#counts&quot;></Link>

<p>The third and final category of browsing data that we keep to facilitate your use of the Analytics feature (should you opt into this) is kept for a year, or, of course, until you choose to clear your Analytics data either for that Device or for your chosen Storage Region on the account.  This is the same data that is stored in all previous scopes if you have Some Analytics enabled (rather than Full Analytics) for the Device in question.</p>
<p>This is simply a count of the number of queries made, the days on which they were made, and whether they were Blocked, Bypassed, or Redirected. This data makes this view possible (available for Devices with either Some or Full Analytics selected).</p>
## <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;data-retention-summary&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-data-retention-summary&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Data Retention Summary</div><Link aria-label=&quot;Skip link to Data Retention Summary&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#data-retention-summary&quot;></Link>

### <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;raw-query-logs&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-raw-query-logs&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Raw Query Logs</div><Link aria-label=&quot;Skip link to Raw Query Logs&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#raw-query-logs&quot;></Link>

<p>Raw DNS query logs are stored for **3 days**. </p>
### <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;statistics&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-statistics&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>Statistics</div><Link aria-label=&quot;Skip link to Statistics&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#statistics&quot;></Link>

<p>Statistics data is stored for:</p>

- **1 month** with hourly time series granularity

- **1 year** with daily time series granularity
