<header id="content-head"><div class="row clearfix"><div class="col-xs-9"># Auto Authorize IP

<div class="excerpt"><div class="rm-Markdown markdown-body" data-testid="RDMD"><p>Learn and log new IPs on a device automatically.</p></div></div></div><div class="col-xs-3">[<i aria-hidden="true" class="icon icon-edit"></i> Suggest Edits](/edit/auto-authorize-ip)</div></div></header><div class="grid-container-fluid" id="content-container"><section class="content-body grid-75"><div dehydrated="# <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;what-is-ip-authorization&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-what-is-ip-authorization&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>What is IP Authorization</div><Link aria-label=&quot;Skip link to What is IP Authorization&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#what-is-ip-authorization&quot;></Link>

<p>This Device Setting controls whether Control D will auto-learn (and log) all source IPs that interact with a Secure DNS resolver. </p>
## <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;how-to-use&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-how-to-use&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>How to Use</div><Link aria-label=&quot;Skip link to How to Use&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#how-to-use&quot;></Link>

<p>Edit any device, and toggle the feature ON. In this mode, when you use your Secure DNS resolvers, newly seen source IPs will be automatically logged and stored alongside the Device. </p>
<span aria-label=&quot;1110&quot; class=&quot;img lightbox closed&quot; role=&quot;button&quot; tabindex=&quot;0&quot;><span class=&quot;lightbox-inner&quot;><Image alt=&quot;1110&quot; loading=&quot;lazy&quot; src=&quot;https://files.readme.io/18c4cfa-ip_auto_learn.JPG&quot; title=&quot;ip_auto_learn.JPG&quot; class=&quot;&quot; width=&quot;100%&quot; align=&quot;&quot; caption=&quot;&quot; height=&quot;auto&quot;/></span></span>
<p>If you have Legacy Resolver enabled, all stored IPs will be able to use your Legacy Resolvers. Disabling this feature will prevent IP logging behavior. </p>
> ## <span class=&quot;callout-icon&quot;>ðŸš§</span><p>Legacy Resolver</p>

<p>If you have Legacy Resolver feature enabled on a Device, Auto IP Authorization cannot be disabled since it requires source IPs to be logged in order for <Link href=&quot;/docs/legacy-resolver&quot; target=&quot;&quot; title=&quot;&quot;>Legacy DNS</Link> to function.</p>
