# Welcome

<Image src="https://files.readme.io/8443d82-controlddefaultbanner.png" alt="Control D Banner" width="1920" />

If you're looking to use the Control D API to interact with the service in scripted/advanced ways, you've come to the right place. The following sections will help you get started.

We strongly encourage you to familiarize yourself with the concepts and [general documentation](https://docs.controld.com/docs) first before using the API.

<Note>
**Use Barry (Experimental)**

Barry, the chatbot found in the bottom right corner of the main website is an API expert, and has complete knowledge of the API docs. Feel free to ask him any question regarding the API. He will happily assist you, and even give you code examples.
</Note>
