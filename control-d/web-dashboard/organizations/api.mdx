<div class="excerpt"><div class="rm-Markdown markdown-body" data-testid="RDMD"><p>Generate tokens to manage your assets using an API.</p></div></div></div><div class="col-xs-3">[<i aria-hidden="true" class="icon icon-edit"></i> Suggest Edits](/edit/org-api)</div></div></header><div class="grid-container-fluid" id="content-container"><section class="content-body grid-75"><div dehydrated="# <div class=&quot;heading-anchor anchor waypoint&quot; id=&quot;what-is-the-api&quot;></div><div class=&quot;heading-text&quot;><div id=&quot;section-what-is-the-api&quot; class=&quot;heading-anchor_backwardsCompatibility&quot;></div>What is the API</div><Link aria-label=&quot;Skip link to What is the API&quot; class=&quot;heading-anchor-icon fa fa-anchor&quot; href=&quot;#what-is-the-api&quot;></Link>

<p>The API section allows a member of an Organization (must be admin or owner) to generate access credentials to be able to use the <Link href=&quot;/reference/get-started&quot; target=&quot;&quot; title=&quot;&quot;>API</Link> in order to automate or script various actions. </p>
<p>Using the API, you can do pretty much everything you can do using the Dashboard (which uses the same API). This includes but not limited to:</p>

- View, create and modify Profiles

- View, create and modify Devices 

- View and modify authorized IP lists

<p>To get started with the API, go to the API section, and click the <button aria-label=&quot;Copy Code&quot; class=&quot;rdmd-code-copy fa&quot;></button>`+` button. Give your token a name, and choose one of 2 types:</p>

- **Read** - Has the ability to view data, but not make any modifications

- **Write** - Can both view and modify data
