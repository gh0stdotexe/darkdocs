# Browser Not Using OS DNS

<Note>
Some browsers may override your system DNS settings with their own DNS-over-HTTPS (DoH) configuration. This guide will help you check and fix this.
</Note>

## 1. Chrome/Chromium

<Steps>
  <Step>Go to `chrome://settings/security`</Step>
  <Step>Scroll down to "Use secure DNS"</Step>
  <Step>Set it to "With your current DNS provider" or disable it</Step>
</Steps>

## 2. Firefox

<Steps>
  <Step>Go to `about:preferences#general`</Step>
  <Step>Scroll down to "Network Settings"</Step>
  <Step>Click "Settings..."</Step>
  <Step>Uncheck "Enable DNS over HTTPS"</Step>
</Steps>

  src="https://files.readme.io/b51ad54-doh_firefox.PNG"

## 3. Best Practices

<Info>
If you want to use DNS-over-HTTPS:
- Configure it in Control D instead of your browser
- This ensures all DNS traffic goes through Control D's filters
- Provides consistent filtering across all applications
</Info>

## 4. Troubleshooting

<Accordion title="Common Issues">
